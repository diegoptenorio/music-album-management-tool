# Setup

1. Install Node.js 18 or higher version

2. Clone reposity:
```
git clone https://github.com/diegoptenorio/music-album-management-tool.git
```
3. Enter project folder:
```
cd music-album-management-tool
```
4. Install dependencies:
```
npm install
```
5. Run Project:
```
npm start
```
